# Scrollspy Demo Page

Update the **Scrollspy demo page** in `src/routes/components/scrollspy/` to showcase all features from the Bootstrap [Scrollspy component](https://getbootstrap.com/docs/5.3/components/scrollspy/).

### Requirements:

- Do **not** create a `ScrollspyPlayground.svelte` component because Scrollspy is an attachment behavior, not a UI component.
- Follow layout used in the Spinner demo page.
- Replicate all example demos from Bootstrap's documentation.
- Apply the `{@attach scrollspy(...)}` directive syntax to target elements or components.
- For each demo, define a callback function and use `attach:scrollspy={...}` bindings for functionality.
- Use `SyntaxHighlighter` to show previewable code snippets.
- Include a detailed **API Reference** documenting the attachment's options and props.
