# Progress Playground

Create a **Progress playground page** in `src/routes/components/progress/` showcasing Bootstrap's [Progress component](https://getbootstrap.com/docs/5.3/components/progress/).

### Requirements:

- Implement `ProgressPlayground.svelte`.
- Follow layout and code organization of the Spinner demo page (`src/routes/components/spinner/`).
- Attempt to replicate each demo shown in the Bootstrap documentation.
- Use `SyntaxHighlighter` component for code snippet previews.
- Include an **API Reference** section that documents all component props and corresponding CSS classes.
