# Nav Component

Create a **Nav component** (located: `src/lib/Nav`) encapsulating all features of the Bootstrap [Nav component](https://getbootstrap.com/docs/5.3/components/navs-tabs/).

### Requirements:

- Compose component parts using dot notation (`Nav.Root`, `Nav.Item`, etc.), similar to the existing Card component.
- Follow patterns with `index.ts`, `types.ts`, and `composition.ts`.
- Include a unit test (`nav.svelte.test.ts`) located in `src/lib/Nav/tests`, referencing the Card component tests.
