# Breadcrumb Component

Create a **Breadcrumb component** (`src/lib/Breadcrumb`) encapsulating Bootstrap [Breadcrumb features](https://getbootstrap.com/docs/5.3/components/breadcrumb/).

### Requirements:

- Compose with dot notation (`Breadcrumb.Root`, `Breadcrumb.Item`).
- Structure elements: `<nav>` with child `<ol>` for items.
- Pattern after Pagination (`src/lib/Pagination`).
- Include unit test (`breadcrumb.svelte.test.ts`) and playground page (`BreadcrumbPlayground.svelte`), referencing Pagination components and tests.
