# Tooltip Playground

Create a **Tooltip playground page** in `src/routes/components/tooltip/` showcasing all the features of the Bootstrap [Tooltip component](https://getbootstrap.com/docs/5.3/components/tooltips/).

### Requirements:

- Implement `TooltipPlayground.svelte`.
- Use Popover demo (`src/routes/components/popover/`) as the structural model.
- Use `SyntaxHighlighter` for code previews.
- Include a detailed **API Reference** section listing props and CSS classes.
