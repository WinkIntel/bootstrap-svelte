# Tabs Component Behavior

Implement the Bootstrap [Tabs JavaScript behavior](https://getbootstrap.com/docs/5.3/components/navs-tabs/#javascript-behavior) in Svelte.

### Requirements:

- Create `nav.svelte.ts` at the root of the Nav component folder.
- Use Svelte reactivity (context API and transitions) similar to the Accordion component.
- Ensure ARIA attributes (`aria-selected`, `aria-labelledby`) are correctly implemented on nav-links and tab-panes.
- Manage show/hide behavior with fade transitions effectively.
